VectorStoreConfig.java file delete kar de!

Spring Boot automatically VectorStore bean create kar deta hai.
Tere application.properties mein configuration hai:

spring.ai.vectorstore.mongodb.collection-name=vector_store
spring.ai.vectorstore.mongodb.vector-index-name=vector_index

Manual configuration ki zarurat nahi hai.

ChromaController mein VectorStore inject ho jayega automatically.
